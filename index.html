Used JavaScript to create a browser based word guessing game. The game uses an array of phrases and write functions to choose a random phrase from that array, split the phrase into letters, and put those letters onto the game board. Each time the player guesses a letter the game compares the guessed letter with the letter in the random phrase and updates the game board accordingly.
